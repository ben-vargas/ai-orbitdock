OrbitDock bundles `ggml-base.en.bin` from this folder for local dictation.

To fetch the model into this folder:
- `make whisper-model`

Expected bundled path in source:
- `OrbitDock/OrbitDock/WhisperModels/ggml-base.en.bin`

Expected bundled path at runtime:
- `WhisperModels/ggml-base.en.bin` inside the app bundle
