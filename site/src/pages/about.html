A cosmic harbor for your AI fleet

OrbitDock started with a simple frustration: running multiple AI coding agents across multiple repos creates chaos. Scattered terminal windows, lost context, missed approval requests, no unified view of what's actually happening.

I don't write most of my code anymore. Agents do. My job shifted to reviewing, managing, and providing guidance at the right time. But there was no tool built for that job. So I built one.

OrbitDock is mission control for AI-assisted product development. It gives you a single dashboard to see every agent, on every machine, across every provider. You can review code, approve tool calls, steer mid-turn, and even let it autonomously dispatch agents against your issue tracker.

Philosophy

What we believe

Server-authoritative

The server is the source of truth. Clients are thin view layers. This means you get the same data and the same experience whether you're on your Mac, your iPhone, or a web browser.

Multi-provider by default

No vendor lock-in. Claude Code and Codex sit side by side, and Codex's OpenAI-compatible API means you can plug in OpenRouter, Qwen, or any other backend. Use whichever model fits the task.

Status at a glance

Five status colors tell you everything. Working, permission needed, question asked, reply waiting, ended. You should never have to read a wall of text to know what needs your attention.

Open source

OrbitDock is free and open source. Your data stays on your machines. Run the server wherever you want. Your laptop, a VPS, a Raspberry Pi.

Under the hood

Built for performance

The stack is simple by design. Fast, self-contained, and easy to deploy.

Rust server

Single binary, zero dependencies, embedded SQLite. Installs in one curl command. Runs on macOS, Linux x86, and Linux ARM (Raspberry Pi).

Native Swift clients

macOS (AppKit) and iOS (UIKit) apps built with SwiftUI. Keyboard-driven on desktop, touch-optimized on mobile. Real native apps, not web wrappers.

WebSocket real-time

Live updates the instant something changes. No polling. Session state, tool calls, and approvals stream in real time.

REST + CLI

Full REST API for everything. The CLI wraps the same API. Script, automate, integrate. However you want to work.
